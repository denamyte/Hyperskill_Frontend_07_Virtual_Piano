Security analyst

As a security analyst, you need to create a simple function that detects potential path traversal patterns in a given file path. The function should return true if the path contains any sequences like ../ indicating directory traversal.


Sample Input:
/uploads/../../../etc/passwd

Sample Output:
true


Sample Input:
/uploads/etc/passwd

Sample Output:
false



Memory limit: 256 MB
Time limit: 15 seconds
