Path building

You are tasked with enhancing a file path building function to prevent path traversal vulnerabilities. Implement a function that takes a base path and a file name, validates the file name, and constructs a secure file path.

If the file name contains .. then log "Error" on the console otherwise construct the secure path as shown in the sample output.

Sample Input:
/uploads
example.txt

Sample Output:
Secure File Path: /uploads/example.txt


Sample Input:
/uploads
../../etc/passwd

Sample Output:
Error



Memory limit: 256 MB
Time limit: 15 seconds
