Utility function

You need to create a utility function that normalizes a file path using Node.js's path module. Ensure that the normalized path is safe and doesn't contain any directory traversal sequences.

Here, in this task, the normalized path will always be safe (prints true) as it is sanitized using the path module.

Sample Input:
/uploads/../../../..//etc/passwd

Sample Output:
Normalized Path: /etc/passwd
Is Safe Path: true



Memory limit: 256 MB
Time limit: 15 seconds
