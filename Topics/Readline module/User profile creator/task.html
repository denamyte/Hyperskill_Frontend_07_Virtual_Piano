User profile creator

Write a program to read information about a user and output the same information. The input will contain the name, age, and the user's favorite programming language in order. Output this information in the following format:

Your name is [name], you are [age] years old, and your favorite programming language is [language].

Use the 'line' event to listen for input and don't output a message for the input!

Sample Input:
Mario 25 Ruby

Sample Output:
Your name is Mario, you are 25 years old, and your favorite programming language is Ruby.



Memory limit: 256 MB
Time limit: 15 seconds
